# P2P 웹 메신저 사용자 매뉴얼

## 1. 개요

이 애플리케이션은 Node.js 기반의 실시간 웹 메신저입니다. 별도의 데이터베이스 없이, 현재 접속 중인 사용자들 간의 실시간 채팅, 파일 전송 등의 기능을 제공합니다.

## 2. 주요 기능

- **로그인:** 사용자는 원하는 이름을 입력하여 메신저에 접속할 수 있습니다.
- **실시간 다중 사용자 채팅:** 온라인 상태인 사용자를 선택하여 그룹 채팅방을 만들 수 있습니다.
- **'입력 중' 알림:** 다른 참여자가 메시지를 입력하고 있을 때 실시간으로 상태가 표시됩니다.
- **사용자 초대:** 이미 생성된 채팅방에 다른 사용자를 추가로 초대할 수 있습니다.
- **파일 전송:** 채팅을 통해 모든 종류의 파일을 주고받을 수 있습니다.
- **채팅방 나가기:** 'X' 버튼을 눌러 참여 중인 채팅방에서 나갈 수 있습니다.
- **멀티라인 메시지:** `Enter` 키로 줄바꿈을, `Shift` + `Enter` 키 또는 '전송' 버튼으로 메시지를 보낼 수 있습니다.

## 3. 시스템 요구사항

- **Node.js:** v14.x 이상 권장

## 4. 설치 및 실행

1.  **의존성 라이브러리 설치:**
    터미널을 열고 이 프로젝트의 루트 폴더에서 다음 명령어를 실행하여 필요한 모든 라이브러리를 설치합니다.
    ```bash
    npm install
    ```

2.  **서버 실행:**
    설치가 완료되면, 다음 명령어로 웹 서버를 시작합니다.
    ```bash
    node server.js
    ```
    서버가 정상적으로 실행되면 터미널에 다음과 같은 메시지가 나타납니다.
    ```
    서버가 3000번 포트에서 실행 중입니다.
    웹 브라우저에서 http://localhost:3000 로 접속하세요.
    ```

## 5. 사용 방법

1.  **로그인:**
    - 웹 브라우저를 열고 `http://localhost:3000` 주소로 접속합니다.
    - 표시되는 로그인 창에 사용할 이름을 입력하고 '참여하기' 버튼을 누릅니다.

2.  **채팅 시작하기:**
    - 로그인 후 왼쪽 사이드바에 현재 접속 중인 사용자 목록이 나타납니다.
    - 함께 채팅하고 싶은 사용자의 이름 옆에 있는 체크박스를 선택합니다.
    - '새로운 채팅 시작' 버튼을 누르면 오른쪽에 새로운 채팅창이 생성됩니다.

3.  **메시지 보내기:**
    - 채팅창 하단의 입력란에 메시지를 입력합니다.
    - 줄바꿈을 하려면 `Enter` 키를 누릅니다.
    - 메시지를 전송하려면 `Shift` + `Enter` 키를 누르거나, '전송' 버튼을 클릭합니다.

4.  **파일 보내기:**
    - 메시지 입력란 옆의 클립(📎) 아이콘을 클릭합니다.
    - 보내고 싶은 파일을 선택하면 즉시 채팅방에 전송됩니다.
    - 다른 참여자는 파일 메시지를 클릭하여 다운로드할 수 있습니다.

5.  **사용자 초대하기:**
    - 채팅창 상단의 '초대' 버튼을 클릭합니다.
    - 현재 채팅방에 없는 다른 사용자 목록이 나타납니다.
    - 초대하고 싶은 사용자를 선택하고 '초대하기' 버튼을 누릅니다.

6.  **채팅방 나가기:**
    - 나가고 싶은 채팅창 상단의 'X' 버튼을 클릭합니다.
    - 채팅방에서 나가게 되며, 해당 채팅창은 화면에서 사라집니다. 다른 참여자에게는 시스템 메시지로 퇴장 사실이 알려집니다.
